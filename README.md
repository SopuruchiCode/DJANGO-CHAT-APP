This is my own spin on a whatsapp inspired messaging app

I used django for both the frontend and backend,
        Redis for real time data handling and short term storage
        SqlLite for long term data backup
        Docker containers for database instances

This project taught me a lot about websockets and how they are used in real world contexts, I implemented them in both Javascript and django
I learnt and utilised the asynchonous functionality of Django in the scope of consumers and routing
